C · SPLIT SIDEBAR
Identity, contacts & résumé pinned left; content scrolls right.
+ Identity always visible
+ App-like, easy to scan
+ Clear identity / work split
– Reads as product UI
– Less calm / editorial
– Sidebar must collapse on mobile
RECOMMENDATION → A · EDITORIAL INDEX
A best matches the brief — calm, elegant, memorable, premium typography — and is the only option that avoids the generic photo-right hero by weaving the portrait into a bio/contact cluster instead of a feature image. It reads as an editorial index, not a marketing page.
B is the safest fallback if the portrait must be the hero (and is easiest to make responsive), but risks feeling template-like. C is strongest for scannability and persistent contact/résumé access — borrow its idea of keeping résumé always reachable — but its sidebar leans product-UI over personal. Proposal: build A, and pull B's clean single-column reflow for mobile + C's persistent résumé affordance.
Selected direction — Home (A), detailed
